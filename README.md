# Almundoweb

Proyecto generado con  [Angular CLI]  version 8.3.6.

## Descargar el proyecto

`git clone https://github.com/marcosbarroso86/almundo-app.git`

## Dependencias

`npm install`

## Server

Ejecute `npm start` Navegue a `http: //localhost:4200/`..
